# Before Assessment

The before assessment establishes the learner's baseline before the custom
project is generated. It should be short enough to finish in one sitting and
broad enough to reveal strengths, gaps, and confidence level.

## How To Use

1. Ask the learner to answer `assessment.md` without using the solution folder.
2. Review answers with `rubric.md`.
3. Summarize baseline strengths and gaps.
4. Choose the project difficulty and scaffolding level.

The score is not a pass/fail grade. It is a starting measurement for comparison
with the after assessment.
