# Before Assessment Rubric

Score each section from 0 to 4.

## Scale

- 0: No answer or unrelated answer.
- 1: Recognizes the topic but cannot apply it.
- 2: Basic attempt with important gaps.
- 3: Mostly correct with minor gaps.
- 4: Clear, correct, and explainable.

## Categories

| Category | Weight | What To Look For |
| --- | ---: | --- |
| SQL basics | 20% | Aggregation, filtering, grouping, date handling |
| SQL debugging | 15% | Join cardinality, duplicate diagnosis, validation queries |
| Python workflow | 15% | File IO, column checks, deduplication, reproducibility |
| Data quality | 15% | Nulls, uniqueness, accepted values, freshness, reconciliation |
| Role-specific thinking | 20% | Clear concepts for target role |
| Explanation | 15% | Can explain problem, data flow, tests, and tradeoffs |
