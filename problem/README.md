# Problem Workspace

This folder contains the custom project generated after intake and before
assessment. The AI mentor should create the actual project brief here instead
of using a fixed, one-size-fits-all assignment.

## Files

- `project-brief.md`: the generated project scenario.
- `assignment.md`: learner-facing requirements.
- `milestones.md`: step-by-step build plan.
- `submission.md`: what the learner submits for review.

## Project Generation Rules

The generated project should:

- Match the learner's target role.
- Use the confirmed tech stack.
- Be challenging but finishable.
- Include realistic messy data or messy requirements.
- Require SQL and Python unless the learner chooses otherwise.
- Include tests or validation checks.
- End with an interview-ready explanation.
