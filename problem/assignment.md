# Assignment Template

The AI mentor should replace this template with a generated assignment.

## Task

Build a data project that solves the scenario in `project-brief.md`.

## Requirements

- Set up a reproducible project environment.
- Create or load the source data.
- Explore and profile the data.
- Build the core SQL and Python workflow.
- Add data quality checks.
- Add tests or validation queries.
- Document how to run the project.
- Prepare interview talking points.

## Constraints

- Use the learner's confirmed stack.
- Keep scope appropriate to the selected difficulty.
- Do not add extra tools unless they teach a useful concept.

## Stretch Goals

The AI mentor should add 1 to 3 stretch goals only after the core project is
complete.
