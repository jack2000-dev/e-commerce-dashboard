# Submission Checklist

When the learner finishes, they should submit:

- Project repository or changed files.
- Commands to run the project.
- Short explanation of the project.
- Screenshots or sample outputs if relevant.
- Tests or validation evidence.
- Known limitations.
- Interview explanation draft.

## AI Review Request Template

Use this prompt when ready for scoring:

```text
Please review my completed workshop project.

Target role:
Difficulty:
Tech stack:
What I built:
How to run it:
What I want feedback on:
Known issues:
```
