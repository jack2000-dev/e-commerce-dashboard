# Milestones Template

The AI mentor should generate 4 to 6 milestones.

## Milestone 1: Project Setup

Deliverables:

- Environment initialized.
- Project folders created.
- Sample data available.

Review questions:

- Why did you choose this structure?
- How does another person run it?

## Milestone 2: Data Understanding

Deliverables:

- Data profile.
- Known quality issues.
- Initial assumptions.

Review questions:

- What is the grain of each dataset?
- What can go wrong with this data?

## Milestone 3: Core Build

Deliverables:

- Main pipeline, model, or analysis.
- SQL and Python workflow.

Review questions:

- What is the transformation logic?
- What are the important joins or aggregations?

## Milestone 4: Validation

Deliverables:

- Tests or quality checks.
- Reconciliation or sanity checks.

Review questions:

- What would break this project?
- How would you catch bad data?

## Milestone 5: Documentation And Interview Prep

Deliverables:

- Final README.
- Architecture explanation.
- Interview talking points.

Review questions:

- Explain the project in two minutes.
- What tradeoffs did you make?
- What would you improve next?
