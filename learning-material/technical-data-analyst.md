# Technical Data Analyst Learning Map

## Core Outcomes

The learner should be able to answer business questions with SQL, Python, and
clear communication.

## Topics

- SQL filtering, joins, aggregation, and windows
- Exploratory data analysis
- Data quality checks
- Metric design
- Cohort or funnel analysis
- Visualization or reporting
- Insight writing
- Assumption tracking
- Recommendation framing

## Project Evidence

- Clear business question
- Correct analytical dataset
- Reproducible analysis
- Quality checks
- Well-explained findings
- Actionable recommendation

## Interview Practice

- Explain how you validated the metric.
- Explain why the analysis answers the business question.
- Explain what could bias the result.
- Explain what decision you would recommend.
